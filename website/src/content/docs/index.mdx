omnifs mirrors external services into your local filesystem. Plan 9 was right,
just 40 years early: **everything is a file**. The world moved to APIs; omnifs
moves it back to paths, for humans and agents alike.

```text
> cd /github/torvalds/linux
> ls
actions  issues  pulls  repo
> cat /dns/cloudflare.com/A
104.16.133.229
> cat /arxiv/papers/1706.03762/metadata.json | jq .title
"Attention Is All You Need"
```

:::caution[Very alpha]
omnifs is early software under active development. The runtime mount is
Linux-only today (macOS and Windows are planned); the CLI runs on macOS and
Linux and drives a Linux container in both cases.
:::
